01-Time

You DAQ module is equipped with a real time clock which needs to be initialized with a valid date and time. The following sample shows how to test if the clock's state is valid and if not, it initializes the DAQ's clock using a DataTime object. The current time is then displayed in the debug output window and looks like this: Time: 09/12/2012 17:57:40.146

using System;
using System.Threading;
using Microsoft.SPOT;
using Microsoft.SPOT.Hardware;
using SecretLabs.NETMF.Hardware.NetduinoGo;
using Nwazet.Go.DAQ;
namespace Tutorials {
    public class Program {
        public static NwazetDAQ daq = new NwazetDAQ();
        public static void Main() {
            daq.Initialize(GoSockets.Socket1);
            var state = daq.GetClockState();
            if (state == NwazetDAQ.ClockState.Invalid) {
                daq.SetDateTime(new DateTime(2012, 09, 12, 17, 57, 00));
            }
            var dt = daq.GetDateTime();
            Debug.Print("Time: " + dt.ToString() + "." + dt.Millisecond);
        }
    }
}
