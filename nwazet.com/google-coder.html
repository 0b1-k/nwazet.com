Google CODER Jetpack for Raspberry Pi

$75.00
Out of stock. Product Discontinued.

Google Coder is a Free Open Source package designed to turn a Raspberry Pi computer into a simple, tiny, personal web server and web-based development environment. It's just what you need for crafting HTML, CSS, and Javascript while you’re learning to code.

There are a ton of resources for learning HTML, CSS, and Javascript: Codecademy and Khan Academy, to name two—and you can use Coder to accompany this instruction as you experiment with making your own programs. When you're ready to get started, read through the included Hello Coder application to acquaint yourself with how a Coder program works.

For those new to coding for the web, it’s a private platform for building web programs. If you’re already a web programmer, Coder can also be a fun sandbox for experimenting with new ideas, or making demos for other people to learn from.


What's in the Nwazet Google CODER Jetpack?

The Nwazet Google CODER Jetpack is pre-assembled and ready to go out-of-the-box with zero configuration required. Just plug it into your network, add power and get coding!

The Jetpack includes:

  • 1x Raspberry Pi Model B
  • 1x Nwazet MiniTux Case, stylish Green or Black design
  • 1x SanDisk 8 GB Class 10 SD Card with Google CODER v4 pre-installed, for super-responsive web sites
  • 1x Ethernet cable, for maximum speed and zero configuration out of the box!
  • 1x 5.5V 1A USB power adapter, perfect for powering the Raspberry Pi
  • 1x 3' foot MicroUSB cable, to connect the Pi to the USB power adapter

For more details on Google CODER, check out: http://googlecreativelab.github.io/coder/
