Mini USB WiFi (802.11b/g/n)

$11.95

Product Type: Networking

When there is no nearby Ethernet connection, you'll need a USB wifi adapter to connect your Raspberry Pi to the Internet or to your local network. This very small adapter will barely stick out, and will be recognized by all our Linux distributions without the need to install any new drivers. We picked this adapter for those cases where a compact project is more important than range. For better range, we recommend our larger USB WiFi adapter.

Raspbian Configuration

To use this WiFi module with the Raspbian distribution on the Raspberry Pi, please follow these steps:

From the console, as root (or using sudo), edit the network interfaces configuration file.

nano /etc/network/interfaces

Edit the file to include a wlan0 interface with the SSID and pre-shared key of the network you want to join.

auto lo

iface lo inet loopback
iface eth0 inet dhcp

allow-hotplug wlan0
auto wlan0
iface wlan0 inet dhcp
wpa-ssid "YourSSID"
wpa-psk "YourKey"

Reboot the Raspberry Pi after saving the updated configuration file.
