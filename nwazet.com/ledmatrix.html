8x8 LED Matrix (Red Square Pixels)


Reg. Price: $3.95
Now $1.00

Product Type: Components, Display Solutions

Product Description

A LED matrix is a great way to start playing with displays: as a simple array of LEDs with simple wiring, it's a system that is easy to understand and use. These LED matrices are controlled by 8 wires for rows, and eight for columns, making it easy to turn any pixel on or off. Pair it with our LED Driver and all you have to do is send 8 bytes through SPI to enjoy a nice and stable 8x8 image.

This LED matrix is the same display that our PIX-6T4 handheld game console kit uses.
